# YujinNotes
A note app

## Features

### User Interface
* Create, view, edit and delete notes
* Sort notes by dragging
* Delete notes by swiping

### Concepts
* MVP Architecture
* Access web server w/ Retrofit
* Unit Testing w/ Mockito
